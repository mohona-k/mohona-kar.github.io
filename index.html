Bootcamp 2020: 3D Solar System

This is the first project I coded for Bootcamp 2020, a 3D abstract of the Solar System. Through this Processing sketch, I have explored the use of classes in order to produce the eight planets in the solar system (not counting Pluto). I also implemented the use of functions to specify several attributes and conditions. For instance, I declared a method called "satellites" which uses the idea similar to a "Parent" and "Child" to output satellites and sub-satellites. This is specified through the use of the variable "Level" which is used in if statements to restrict showing an infinite number of these "children". Moreover, I also showcase utilisations of for loops, 3D graphics and coding decisions with conditionals.
